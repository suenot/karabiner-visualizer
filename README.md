# karabiner-visualizer

Next.js frontend that renders [Karabiner-Elements](https://karabiner-elements.pqrs.org/) `complex_modifications` rule JSON files as interactive SVG keyboard layouts.

Companion repo: [suenot/karabiner](https://github.com/suenot/karabiner) — the actual rule configs being visualized.

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS
- SVG-based keyboard rendering

## Getting started

```bash
npm install
npm run dev
```

Open <http://localhost:3000>.

## Project status

Scaffolding. UI for loading rule JSON and rendering SVG keymaps will be added incrementally.
